### 🚀 Featured Portfolio Projects

Here is a selection of hands-on data science and machine learning projects that reflect my technical progress:

#### 1. [Titanic Challenge: ML Model Duel 🚢]([https://github.com/Lucas-C-T/titanic-survival-prediction](https://github.com/Lucas-C-T/projeto-titanic-duelo-modelos-ebac-M41))
* **Objective:** Achieve the best accuracy on the famous Kaggle Titanic competition by designing, optimizing, and dueling multiple classification models.
* **Techniques Used:** Outlier detection, missing data imputation, Class Balancing (SMOTE), standardization, and Dimensionality Reduction (PCA).
* **Models Compared:** Support Vector Classifier (SVC), Random Forest, and XGBoost optimized via `GridSearchCV`.
* **Outcome:** Mastered model evaluation metrics and addressed overfitting challenges, proving how model tuning drastically boosts real-world Kaggle test performance.

#### 2. [Customer Purchase Intent Prediction 📊]([https://github.com/Lucas-C-T/customer-purchase-intent](https://github.com/Lucas-C-T/previsao-intencao-compra-ecommerce-ebac-M37))
* **Objective:** Predict whether an e-commerce visitor will complete a purchase based on demographic data and customer behavioral tracking.
* **Techniques Used:** Structured Exploratory Data Analysis (EDA), business storytelling with visual charts, data cleaning, and feature scaling.
* **Models Compared:** Logistic Regression vs. Random Forest Classifier.
* **Outcome:** Random Forest yielded a massive improvement (~91% accuracy and 95% recall for active buyers), significantly reducing false negatives to help marketing teams target hot leads efficiently.
